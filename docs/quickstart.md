# Quickstart — aurekai-gitpod

Open this repo in Gitpod to get an Aurekai-ready cloud dev environment.

[![Open in Gitpod](https://gitpod.io/button/open-in-gitpod.svg)](https://gitpod.io/#https://github.com/aurekai/aurekai-gitpod)

## What's included

- `akai` CLI pre-installed at `0.8.0-alpha.4`
- Auto-run `akai doctor --deep` on workspace start
- Python extension in VS Code

## Validate

```bash
bash tests/validate-schemas.sh
bash tests/validate-scripts.sh
```
